# 🛒 E-Commerce(Shop Now) React Native App

A modern React Native E-Commerce(Shop Now) application built using React Native CLI. The app demonstrates product browsing, cart management, favorites, filtering, sorting, search, and a clean UI following best development practices.

---

## 📱 Features

### Home Screen
- Product listing from DummyJSON API
- Search products
- Category filter
- Price sorting (Low → High / High → Low)
- Skeleton loading
- Pull to refresh
- Premium product cards

### Product Details
- Product image
- Title
- Description
- Brand
- Rating
- Price & Discount
- Warranty Information
- Return Policy
- Reviews
- Add to Cart
- Add to Favorites

### Cart
- Add / Remove products
- Increase / Decrease quantity
- Stock validation
- Minimum quantity validation
- Total price calculation
- Total item count

### Favorites
- Add / Remove favorite products
- Move products to Cart

### Other Features
- Redux Toolkit State Management
- Axios API Integration
- React Navigation
- Toast Messages
- Poppins Font
- Responsive UI
- Clean Folder Structure

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Dee-co/e-commerce-app
```

Install dependencies

```bash
npm install
```

### Android

```bash
npx react-native run-android
```

### iOS

```bash
cd ios
pod install
cd ..
npx react-native run-ios
```
